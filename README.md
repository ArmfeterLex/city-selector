# City Selector

Консольное приложение на C# для выбора города из списка по номеру или первой букве.

## Функциональность

- Список городов: Брест, Витебск, Гомель, Минск, Полоцк
- Выбор по номеру (1-5)
- Выбор по первой букве названия
- Защита от некорректного ввода
- Бесконечный цикл

## Пример работы

Введите первую букву города или номер (1-5):
3
Выбран город: Гомель

Введите первую букву города или номер (1-5):
Б
Выбран город: Брест

Введите первую букву города или номер (1-5):
6
Ошибка: Некорректный номер города.

## Куда можно встроить

- В справочные системы
- В меню выбора
- В обучающие приложения

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Программа работает бесконечно, выход через Ctrl+C. При вводе буквы выбирается первый подходящий город.
